## 是什么
一个极简的 AI Agent, 使用 python + OpenAISDK + Deepseek 实现

参考文章：
[如何从零开始实现一个 AI Agent 框架（理论+实践）](https://mp.weixin.qq.com/s/z1aDaPFhjYb2cv-9bFbdaQ)

## 什么能力

1. 基于deepseek chat 的聊天能力
2. Agent拓展能力
    1. 执行shell命令
    2. 读取文件
    3. 写入文件
    4. 执行python代码

## 怎么使用

1. 配置deepseek apikey环境变量 DEEPSEEK_API_KEY
2. 运行main.py
3. 在控制台与Agent互动
